# GridFire Saga

A 2D top-down wave shooter made with Unity 6.3 LTS.

## Controls
- WASD: Move
- Mouse: Aim
- Left Click: Shoot
- R: Reload
- Space: Dash
- Esc: Pause

## Notes
This ZIP contains the C# source structure. Create/open a Unity 2D project, then copy the `Assets/Scripts` folder into your Unity project `Assets/` folder.
